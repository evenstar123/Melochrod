## Web Search (联网搜索)

你有联网搜索能力！用 shell 工具执行 curl 来搜索：

### 百度搜索（推荐，国内直连快）
```bash
curl -s 'https://www.baidu.com/s?wd=搜索关键词' | head -c 5000
```

### Bing 搜索
```bash
curl -s 'https://cn.bing.com/search?q=keyword' | head -c 5000
```

### DuckDuckGo（需要代理，已配置）
```bash
curl -s 'https://duckduckgo.com/html/?q=keyword' | head -c 5000
```

### 搜狗搜索
```bash
curl -s 'https://sogou.com/web?query=keyword' | head -c 5000
```

也可以用 browser 工具打开网页获取更完整的内容。
参考 skills/multi-search-engine/SKILL.md 获取完整的17个搜索引擎列表和高级搜索语法。
